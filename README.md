# JooHana Theme ✦

[joohana.net](https://joohana.net)에서 사용할 텀블러 테마입니다.

이 프로젝트는 로컬에서 실시간으로 텀블러 테마를 미리보며 편집할 수 있도록 개발 환경이 구축되어 있습니다.

---

## 🛠️ 개발 시작하기

### 1. 패키지 설치
프로젝트 루트 폴더에서 필요한 개발 의존성 패키지를 설치합니다.
```bash
npm install
```

### 2. 로컬 미리보기 서버 실행
아래 명령어를 실행하여 로컬 미리보기 대시보드 서버를 켭니다.
```bash
npm run dev
```

서버가 구동되면, 터미널 로그에 표시되는 주소(기본값: [http://localhost:3000](http://localhost:3000))로 접속하여 실시간 대시보드를 사용할 수 있습니다.

### 3. 기능 및 특징
- **실시간 우회 매핑**: 테마 파일(`src/theme.html`) 내부의 jsDelivr CDN 경로(`.../src/style.css`, `.../src/script.js`)가 로컬 폴더 안의 실제 파일(`src/style.css`, `src/script.js`)로 실시간 매핑되어 서빙됩니다. 복잡한 외부 프록시 툴이나 크롬 익스텐션을 사용할 필요가 없습니다.
- **테마 튜닝 대시보드**: 뷰포트 크기 변경 및 텀블러 커스텀 테마 변수를 대시보드 상에서 실시간으로 조작하여 반응형 스타일을 다각도로 테스트해볼 수 있습니다.
- **다양한 목 데이터 시나리오**: 텍스트, 이미지, 비디오, 질문 답변 등 7가지 이상의 다채로운 포스트 타입으로 반응형 레이아웃의 완성도를 확인합니다.
- **목 데이터 자유 커스텀**: 프로젝트 루트에 `tumblr-mock.json` 파일을 작성해 본인만의 블로그 및 포스트 정보로 목 데이터를 커스텀하여 테마에 대입해 볼 수 있습니다.
